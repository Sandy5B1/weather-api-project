Weather API Project 🌦️

📌 Description

This project is a simple Python application that fetches real-time weather data using an API.
The user enters a city name, and the program displays temperature and weather conditions.

---

⚙️ Features

- Uses API to get live weather data
- Accepts user input (city name)
- Displays temperature in Celsius
- Shows weather description
- Handles errors (invalid city name)

---

🛠️ Technologies Used

- Python
- Requests library
- Weather API

---

▶️ How to Run

1. Install required library:
   pip install requests

2. Replace "YOUR_API_KEY" in the code with your API key

3. Run the program:
   python weather.py

4. Enter city name when asked

---

Temperature: 32°C
Weather: clear sky
